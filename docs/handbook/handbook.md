# **AI Vibe Coding**

## **What Is Vibe Coding?**

**Vibe Coding** is a fast, conversational way of creating digital products with AI.

Instead of starting by writing every line of code manually, you describe what you want to build in natural language. AI coding tools can help generate interfaces, logic, databases, workflows, and iterations. You remain responsible for the problem, the product decisions, testing, and the user experience.

For this Mini Hackathon, Vibe Coding can help teams quickly create:

- Websites and landing pages
- Lightweight web applications
- Interactive tools or dashboards
- AI-powered assistants
- Community services and workflow tools
- Data visualisations
- Early-stage product prototypes

The focus is not on producing the most complex code. The focus is on creating something that clearly addresses a real problem and can be meaningfully experienced.

## **6.1 Recommended Structure**

**Top Section**

- Project name
- Short tagline
- Team name / team members

**Middle Section**

- The problem
- Target users
- Your solution
- How AI is used
- Key screenshots, diagrams, or product flow

**Bottom Section**

- Why it matters
- Current stage of the prototype
- Next step or collaboration needs
- QR code linking to the live demo, video, or project page

## **6.2 Design Principles**

- Keep text concise and visual hierarchy clear.
- Use large, readable headings.
- Show the product, not only the idea.
- Include screenshots or illustrations of the actual prototype.
- Avoid overcrowding the poster with long paragraphs.
- Make it easy for a visitor to ask: “Can you show me how this works?”
- Use a QR code wherever possible.
